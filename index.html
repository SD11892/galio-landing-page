What is Galio?

Free & Open Source

Galio Framework is a 100% free and open source project, licensed under MIT. It will always remain free to use, powered by a massive world-wide community.

Beautifully designed

Clean, simple, and functional. With ready-made components, typography, and a gorgeous base theme that adapts to each platform, you'll be building in style.

Installation

Include Bootstrap's source Sass and JavaScript files via npm, Composer or Meteor. Package managed installs don't include documentation, but do include our build system and readme.

npm install bootstrap
gem install bootstrap -v 4.0.0

Road Map

We have great plans and a long road ahead of us. Our goal is to develop a fully customisable mobile framework based on react native.

July 2018

Idea

  • Research React native limitations & possibilities
  • Research how apps are developed so we can ease the process by unifying or simplyfing different processes
  • Building first version of a design system intended to be used as a base to start from, for any future project
August 2018

Public launch @ GitHub

  • Reach to existing React native communities and spread the word about the new experimental solution
  • Build new communities on Medium, Reddit, Facebook Groups, GitHub and more
  • Making sure we get feedback and start working on reaching a stable basic version
